
# k-DPP4Ranking

A PyTorch implementation of LkP

# Requirements
* Python 2 or 3
* [PyTorch v0.4+](https://github.com/pytorch/pytorch)
* Numpy
* SciPy

# Usage
1. Install required packages.
2. use 3_item_dpp_emb.py to create the pre-learned diverse kernel
3. run <code>python LkP.py</code>
